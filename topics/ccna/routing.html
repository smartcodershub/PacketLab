Routing Protocols

How routers learn paths and forward packets — static, RIP, OSPF, EIGRP, BGP

Types of Routing

  • Static routing — manually configured. Best for small or stub networks. No overhead but no automatic failover.
  • Dynamic routing — routers share route information automatically. Adapts to network changes.

Administrative Distance Rule

Lower AD = more trusted. If two protocols know a route, the one with lower AD wins.

Static Route Configuration

Router(config)# ip route 192.168.2.0 255.255.255.0 10.0.0.2
# Destination network   Mask              Next-hop IP

Router(config)# ip route 0.0.0.0 0.0.0.0 10.0.0.1
# Default route (gateway of last resort)

OSPF Basic Configuration

Router(config)# router ospf 1
Router(config-router)# network 192.168.1.0 0.0.0.255 area 0
Router(config-router)# network 10.0.0.0 0.0.0.3 area 0

# Verify
Router# show ip ospf neighbor
Router# show ip route ospf
