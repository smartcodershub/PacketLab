File System & Navigation

Linux directory structure and essential navigation commands

Linux Directory Structure

PathPurpose
/Root — top of the filesystem tree
/homeUser home directories
/etcSystem configuration files
/varVariable data — logs, databases
/usrUser programs and libraries
/binEssential binaries (ls, cp, mv)
/tmpTemporary files — cleared on reboot
/devDevice files

Essential Commands

pwd                    # Print current directory
ls                     # List files
ls -la                 # List all files with details + hidden
cd /home/user          # Change to absolute path
cd ..                  # Go up one level
cd ~                   # Go to home directory
mkdir mydir            # Create directory
mkdir -p a/b/c         # Create nested directories
rmdir mydir            # Remove empty directory
rm -rf mydir           # Remove directory and contents (CAREFUL)
cp file.txt /tmp/      # Copy file
mv file.txt newname    # Rename / move file
cat file.txt           # Display file contents
less file.txt          # Page through file (q to quit)
head -20 file.txt      # First 20 lines
tail -f /var/log/syslog  # Follow log in real time

Finding Files

find / -name "*.conf"           # Find all .conf files from root
find /home -type f -size +1M   # Files larger than 1MB
find . -mtime -7                # Files modified in last 7 days
which python3                   # Find location of a command
locate nginx.conf               # Fast file search (uses index)
