NAT & PAT

Translate private IPs to public for internet access — static NAT, dynamic NAT, PAT overload

What is NAT?

NAT (Network Address Translation) converts private IP addresses to a public IP before sending traffic to the internet.

TypeDescriptionRatio
Static NATOne private → one public IP1:1
Dynamic NATPool of public IPs, assigned as neededMany:many
PAT (Overload)Many private → one public IP, differentiated by portMany:1

PAT Configuration (Most Common)

! Define which IPs to translate
Router(config)# access-list 1 permit 192.168.1.0 0.0.0.255

! Enable PAT on outside interface
Router(config)# ip nat inside source list 1 interface gi0/0 overload

! Mark interfaces
Router(config)# interface gi0/1
Router(config-if)# ip nat inside

Router(config)# interface gi0/0
Router(config-if)# ip nat outside

! Verify translations
Router# show ip nat translations
Router# show ip nat statistics

Static NAT (Server accessible from internet)

! Map private server 192.168.1.10 → public 203.0.113.10
Router(config)# ip nat inside source static 192.168.1.10 203.0.113.10
