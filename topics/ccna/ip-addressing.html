Private IP Ranges (RFC 1918)

  • 10.0.0.0 – 10.255.255.255 → Class A private
  • 172.16.0.0 – 172.31.255.255 → Class B private
  • 192.168.0.0 – 192.168.255.255 → Class C private

Subnetting Example

Given: 192.168.1.0/26

  • Subnet mask: 255.255.255.192
  • Network: 192.168.1.0
  • Broadcast: 192.168.1.63
  • Usable hosts: 192.168.1.1 – 192.168.1.62 (62 hosts)

IPv6 Basics

2001:0db8:85a3:0000:0000:8a2e:0370:7334   # Full IPv6
2001:db8:85a3::8a2e:370:7334              # Compressed (drop leading zeros + :: for consecutive zeros)

fe80::/10    # Link-local
fc00::/7     # Unique local (like private IPv4)
2000::/3     # Global unicast
